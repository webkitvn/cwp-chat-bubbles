# CWP Chat Bubbles

A modern, standalone WordPress plugin that auto-injects floating chat bubbles for 8+ messaging platformsâ€”no dependencies, no shortcode required. Designed for Asian and global markets, with native QR code support, mobile-first design, and full WordPress standards compliance.

---

## Plugin Features
- **Auto-injects chat bubbles** on all frontend pages (no shortcode needed)
- **Supports 8+ platforms**: Phone/Hotline, Zalo, WhatsApp, Viber, Telegram, Facebook Messenger, Line, KakaoTalk
- **QR code integration** for each platform
- **Comprehensive admin options**: enable/disable plugin, customize appearance, manage platforms, upload QR codes
- **Drag-and-drop sorting** for platform order

---

## Installation
1. Upload the plugin folder to `/wp-content/plugins/cwp-chat-bubbles/` or install via WordPress admin
2. Activate the plugin through the 'Plugins' menu
3. Visit **Chat Bubbles** in the WordPress admin menu to configure settings

### Requirements
- WordPress 5.0+
- PHP 7.4+

---

## Usage
- **Frontend**: Chat bubbles appear automatically in the bottom-right corner of every page
- **Admin**: Configure platforms, upload QR codes, customize appearance, and manage advanced options via the **Chat Bubbles** menu
